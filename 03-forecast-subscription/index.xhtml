Manage Forecast Subscriptions

Welcome to the Forecast Subscriptions. This demo uses w3c XForms and has been tested with FireFox 2.0 with the XForms addon.

This demo allows you to manipulate contacts using XForms and eXist using basic database operations CRUD (Create, Read/Search, Update, Delete) operations:

  1. List all Subscriptions
  2. Create New Subscriptions
  3. Search Subscriptions
